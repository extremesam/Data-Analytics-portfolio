# 🔁 Customer Churn Analysis Dashboard

## 📊 Overview

This project focuses on analyzing customer churn behavior to understand why customers leave and how retention can be improved.

It helps businesses improve customer lifetime value and reduce revenue loss.

---

## 🎯 Objectives

- Measure churn rate
- Identify factors influencing churn
- Segment customers based on behavior
- Improve customer retention strategies

---

## 📌 Key Metrics

- Total Customers
- Churn Rate
- Retention Rate
- Active Customers
- Average Customer Lifetime Value

---

## 📊 Visualizations

- Churn by Contract Type
- Churn by Payment Method
- Churn by Age Group
- Customer Tenure Distribution
- Service Usage vs Churn
- Risk Segmentation Analysis

---

## 🛠️ Tools & Technologies

- Power BI

- Data Modeling

## 📈 Key Insights

- Short-term contracts have higher churn rates
- Certain payment methods are linked to higher churn
- Customers with lower tenure are more likely to leave
- Service usage patterns strongly influence retention
