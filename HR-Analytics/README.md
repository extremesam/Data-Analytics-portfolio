# 👥 HR Analytics Dashboard

## 📊 Overview

This project provides an interactive HR analytics dashboard built in Power BI to analyze workforce structure, employee attrition, and organizational health.

It helps HR teams and management understand key workforce trends and make data-driven decisions around retention, hiring, and employee satisfaction.

---

## 🎯 Objectives

- Analyze employee attrition patterns
- Understand workforce demographics
- Evaluate salary distribution across departments
- Identify high-risk attrition groups
- Support HR decision-making with KPIs

---

## 📌 Key Metrics

- Total Employees
- Attrition Rate
- Average Age
- Average Salary
- Department Headcount
- Gender Distribution

---

## 📊 Visualizations

- Attrition by Department
- Attrition by Age Group
- Employee Distribution by Gender
- Salary Distribution Analysis
- Job Role Breakdown
- Years at Company vs Attrition

---

## 🛠️ Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Excel (Data Source)
